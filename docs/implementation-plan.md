# TrackIT Mobile — План реализации

## Референс дизайна
- Стиль: Trello-подход (горизонтальный Kanban)
- Визуальный референс: https://dribbble.com/shots/20973715-ToDo-App-Kanban-Board-Gantt-chart

---

## Фаза 1: Основа (MVP) — 3-5 дней

### 1.1 Навигация и авторизация ✅ DONE
- [x] Expo Router (file-based routing)
- [x] 4-табовая навигация (Bottom Tabs)
- [x] Логин через Keycloak (email/password → API token)
- [x] SecureStore для хранения токена
- [x] Автологин при повторном запуске

### 1.2 Список проектов ✅ DONE
- [x] Загрузка проектов из V1 API
- [x] Pull-to-refresh
- [x] Навигация в проект

### 1.3 Kanban-доска
- [x] Горизонтальный скролл колонок (snap)
- [x] Вертикальный скролл карточек в колонке
- [x] Карточки: ID, заголовок, приоритет
- [ ] Карточки: labels, due date, comments count, assignee avatars
- [ ] Переключатель Kanban ↔ List view
- [ ] Header с названием проекта

### 1.4 Детали задачи
- [x] Просмотр: заголовок, статус, приоритет, описание, дата
- [ ] Properties card (статус, приоритет, assignee, labels)
- [ ] Комментарии (загрузка + отправка)
- [ ] SLA индикаторы
